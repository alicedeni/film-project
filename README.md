# Film Project

SPA на **React + TypeScript + Vite**: каталог фильмов по данным [ПоискКино API](https://poiskkino.dev)

## Требования

- Node.js **18+**
- npm

## Установка и запуск

```bash
npm install
```

1. Скопируйте `.env.default` в `.env`

2. Заполните `VITE_KINOPOISK_API_KEY`

3. Запуск в режиме разработки:

   ```bash
   npm run dev
   ```

4. Сборка:

   ```bash
   npm run build
   ```

5. Локальный просмотр сборки:

   ```bash
   npm run preview
   ```
